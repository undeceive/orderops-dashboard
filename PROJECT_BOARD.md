# OrderOps Dashboard Project Board

OrderOps Dashboard is a marketplace operations dashboard for tracking products, orders, marketplace listings, inventory differences, price mismatches, image issues, and other problems across channels such as Walmart, Shopify, Amazon, eBay, Zoro, TopDawg, and Temu.
